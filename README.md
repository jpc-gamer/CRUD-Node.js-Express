# CRUD-Node.js-Express

Este projeto é um sistema CRUD desenvolvido com Node.js e Express no backend, responsável por gerenciar usuários através de operações de criação, leitura, atualização e exclusão.

O frontend consome a API por meio de requisições HTTP (GET, POST, PUT e DELETE), permitindo a interação com o banco de dados de forma dinâmica.

## Como executar

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
cd seu-repositorio
npm install
node server.js
